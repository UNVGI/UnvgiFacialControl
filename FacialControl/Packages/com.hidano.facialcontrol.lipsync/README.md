# com.hidano.facialcontrol.lipsync

`com.hidano.facialcontrol.lipsync` は、FacialControl コアパッケージと `com.hidano.ulipsync-asio` を接続する uLipSync 連携アダプタパッケージです。

## 対応環境

- Unity 6000.3.2f1 以降
- Windows Standalone / Editor
- `com.hidano.facialcontrol`
- `com.hidano.ulipsync-asio`

## パッケージ内容

- `Runtime/`
- `Editor/`
- `Tests/EditMode/`
- `Tests/PlayMode/`
- `Tests/Shared/`
- `Samples~/MicLipSyncDemo/`
- `Documentation~/`

詳細な利用手順、Samples の二重管理ルール、旧 `uLipSyncBlendShape` 手結線からの移行手順は後続タスクで `Documentation~/` に追加します。
