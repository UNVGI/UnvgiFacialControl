## Project Description (Input)
InputAction と Expression の紐付けを ScriptableObject (InputBindingProfileSO) で永続化し、Unity Editor の Inspector から GUI でキーコンフィグを設定可能にする。FacialInputBinder MonoBehaviour が Profile を読んで内部で InputSystemAdapter にバインディング登録する。サンプルシーンを TestExpressionToggle (コード直書き) から FacialInputBinder + SampleInputBinding.asset 方式へ更新し、preview.1 リリースに含める。要件は docs/requirements-qa.md Q8-3、設計と作業内容は docs/work-procedure.md P23 を参照。
